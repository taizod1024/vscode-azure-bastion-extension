# Azure Bastion Extension (Work In Progress)

## Usage

1. Open a terminal
   1. Install Azure CLI `winget install Microsoft.AzureCLI`
   1. Install SSH Extension `az extension add -n ssh`
2. Open VS Code
   1. Install the Azure Bastion extension
   2. Configure the `azure-bastion` settings
   3. Click the `Azure Bastion` button in the status bar
   4. Connect via SSH
